# 🛒 Quantium Retail Analytics: Chips Category Optimization

## 📖 Project Overview
This project is part of the Quantium Data Analytics Virtual Experience Program. The goal was to analyze transaction and customer data to identify high-value demographics and evaluate the performance of a new store layout through A/B testing. 

## 🛠️ Tech Stack & Tools
* **Programming:** Python (Pandas, NumPy)
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebooks
* **Reporting:** Microsoft PowerPoint (Pyramid Principle Framework)

## 📊 Key Insights & Deliverables
### 1. Customer Segmentation (Task 1)
* Cleaned and engineered a dataset of over 260,000 retail transactions.
* **Finding:** Mainstream Young Singles/Couples are the most profitable demographic, showing a high willingness to pay for premium 175g chip packets.

### 2. A/B Testing & Store Trial Assessment (Task 2)
* Utilized Pearson correlation to match trial stores (77, 86, 88) with statistically similar control stores based on pre-trial total sales and customer count.
* **Finding:** The new store layout in Stores 77 and 86 drove a statistically significant spike in sales during the trial period.

### 3. Commercial Presentation (Task 3)
* Synthesized technical findings into an executive-level slide deck using the Pyramid Principle.
* Recommended a chain-wide rollout of the new layout and a reallocation of premium shelf space targeting the high-value demographic.
